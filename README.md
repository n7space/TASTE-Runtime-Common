## Usage

Update submodules:  
`git submodule update --init --recursive`

Create build directory:  
`cd ..`  
`mkdir TASTE-Runtime-Common-build`  
`cd TASTE-Runtime-Common-build`  

Use CMake to generate build files:  
`cmake ../TASTE-Runtime-Common-build`

Build the project:  
`make`
